文档和代码功能划分和注意事项

  • 文档专注于 OS 的原理和同时涉及 OS 和 Rust 特性的 Rust 语言特性，不会花篇幅来讲解 Rust 语言本身；

  • 代码中书写的注释专注于功能性，不会有流程性和原理性的介绍；
  • 文档应有全面的包括流程性、原理性和功能性的介绍。
